# 🎮 Video Game Sales Data Visualization

## 📌 Overview

This project explores and visualizes global video game sales data using R. The analysis focuses on identifying trends across platforms, genres, publishers, and regions through data visualization techniques.

The project demonstrates practical skills in:

* Data analysis
* Data cleaning
* Data visualization
* Exploratory Data Analysis (EDA)
* R programming

# ▶️ How to Run the Project

## 1. Clone the Repository

```bash
git clone https://github.com/bharadwajagraharam-98/DATA-VISUALIZATION.git
```

## 2. Navigate to the Project Folder

```bash
cd DATA-VISUALIZATION
```

## 3. Install Required Packages

```r
install.packages(c("ggplot2", "dplyr", "tidyr", "readr"))
```

## 4. Run the Script

```bash
Rscript visualization.R
```
